boolean trasla ( Traslazione...  traslazioni)
private

Trasla, se possibile, ogni singolo quadrato della traslazione con indice associato.

Per ogni quadrato della figura, viene cercata la traslazione corrispondente nel vettore in base all'indice numerico.

Parameters
traslazioniVettore di oggetti Traslazione che deve avere la stessa dimensione del numero di quadrati che compongono la figura
Returns
true se la traslazione è avvenuta con successo
Exceptions
RuntimeExceptionse il numero di oggetti translazione non è uguale al numero di quadrati
