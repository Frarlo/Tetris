◆ leggiTasto()

int leggiTasto ( )

Restituisce un intero che rappresenta la scelta dell'utente.

Il valore restituito è compreso tra 0 a 4, che rispettivamente indicano:

  • -1 - Nessun input NO_INPUT
  • 0 - Spostamento a destra SPOSTA_DESTRA
  • 1 - Spostamento a sinistra SPOSTA_SINISTRA
  • 2 - Spostamento in giù SPOSTA_GIU
  • 3 - Rotazione a destra RUOTA_DESTRA / Nuova partita nel menu iniziale NUOVA_PARTITA
  • 4 - Rotazione a sinistra RUOTA_SINISTRA
  • 5 - Hard drop DROP

Questo metodo non deve bloccare l'esecuzione del thread, ovvero non deve aspettare la presenza effettiva di input, ma ritornare un input se è presente o NO_INPUT altrimenti.

Returns
input dell'utente tra quelli sopra specificati
